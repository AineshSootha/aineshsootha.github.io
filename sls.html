ServerlessManager

A tool that simplifies deployment of Serverless Lambda Functions

A project I built at BestX (State Street)

Github   PyPi

pip install slsmanager

Get Started

• Set up an IAM user with the provided policy ->

• Set up a CodeCommit Repository using slsmanager/AWS Console

• Build all resources using the provided Cloudformation Template ->

• Place the provided buildspec file -> in your repository

• Place the provided gitDiff script -> in your repository


That's It!

Now, whenever you push new commits to your repository, new lambda functions will be created, and with subsequent deploys, they will be updated and redployed!
